GemaDigital Laravel Framework

## Comands

Install the project
``` bash
$ php artisan install
```

Run the project (using env information)
``` bash
$ php artisan run
```

Packages a ready for production zip
``` bash
$ php artisan package
```

Publishes the framework assets
``` bash
$ php artisan publish
```

## Installation

Via Composer

``` bash
$ composer require gemadigital/framework
```
